# 小鹏汽车（XPEV / 9868.HK）小店分析报告

> 分析日期：2026年5月5日 | 数据基准：2025年年报（2026年3月20日披露）

---

## 【第一步】这家店卖什么？

### 一句话概括

**小鹏汽车是一家以"智能化"为核心卖点、正在从亏损走向盈利的中国新能源车企，2025年交付量翻倍至43万辆，跃居新势力前三。**

### 收入构成（2025年年报）

| 业务板块 | 营收（亿元） | 占比 | 同比增长 | 一句话理解 |
|----------|:---------:|:----:|:------:|-----------|
| **汽车销售** | 683.79 | 89.1% | +90.8% | **卖车的钱**——主力是MONA M03（11-14万）、P7+（19-21万） |
| **服务及其他** | 83.41 | 10.9% | +65.6% | **技术输出+充电+维保**——含与大众的合作研发收入 |

### 谁在买？

| 客群画像 | 代表车型 | 价位 | 核心诉求 |
|----------|---------|:----:|------|
| **年轻首购族** | MONA M03 | 12-14万 | 性价比高、智能化好、颜值在线 |
| **中产家庭** | P7+、G6 | 18-25万 | 空间大、智驾强、品牌不low |
| **多孩/商务** | X9（纯电+增程） | 35-42万 | 大空间、豪华感、增程无里程焦虑 |
| **海外用户** | G6、G9为主 | — | 中国智能电动车的高性价比替代 |

### 商业模式判断

**卖产品 + 技术输出**——不仅是卖车，还向大众汽车输出智能驾驶技术（联合开发车型、收取技术服务费），这是与传统车企最大的不同。

### 必问三个问题

1. **谁在买？** — 主要是中国20-40岁城市中产，2025年靠MONA M03打入10-15万主流市场后客群大幅拓宽。
2. **为什么买小鹏不买比亚迪/特斯拉？** — **智能化体验**是小鹏最大的差异化。XNGP智驾系统在中国路况的体验被公认为第一梯队，语音助手"小P"也是行业标杆。
3. **能不能不买？** — 消费者可选择的车太多了（比亚迪、特斯拉、小米、理想、蔚来...），**小鹏的定价权很弱**，只能靠产品力和智能化打动用户，没有品牌溢价。

毛利率：18.9%  （2024年：14.3%，+4.6个百分点）

### 口诀验证

- **毛利率看赛道** → 18.9%，在汽车行业属于中上水平（传统车企10-20%，豪华品牌20-30%）

## 【第三步】这家店靠不靠谱？

### 利润质量检验

```
经营现金流 / |净亏损| = 82.6亿 / 11.4亿
→ 虽然仍在亏损，但经营现金流已经大幅转正
→ 自由现金流 = 82.6 - 33.5 ≈ +49.1亿 ✅
```

> ✅ **2025年是小鹏的经营分水岭**：经营现金流从2024年的-20.1亿转为+82.6亿，自由现金流从-44.4亿转为+49.1亿。**店还在亏钱，但已经能自己造血了。**

### 应收账款检验

```
应收账款：约41.2亿元（占总资产4%）
应收增速 vs 营收增速：基本匹配
```

> 汽车行业以现款/按揭销售为主，应收账款主要来自技术服务和B端业务，占比不高，风险可控。

### 存货检验

```
存货：约63.5亿元（主要是成品车和在产车）
存货周转天数：约35天
```

> 作为车企，35天库存周转属于非常健康水平（行业优秀标准<45天），说明MONA系列产品动销良好。

### 资本消耗

```
资本开支：33.5亿  |  自由现金流：+49.1亿
→ 扩张不需要"借钱烧"，自己赚的现金流就够了 ✅
```

> 33.5亿资本开支主要用于超快充网络建设、新车型模具和产线改造。相比于公司767亿的营收体量，资本开支强度不高。

## 【第四步】这家店的护城河

### 护城河评级：★★★（中等，正在构建中）

| 护城河类型 | 强度 | 证据 |
|-----------|:----:|------|
| **技术壁垒（智驾）** | ★★★★ | XNGP全场景智驾国内公认第一梯队，VLA大模型架构行业首发。720亿参数世界基座模型、图灵芯片自研。 |
| **技术输出（变现能力）** | ★★★ | 向大众汽车输出EEA电子电气架构和智驾技术，2025年服务收入83.4亿。是国内唯一靠卖技术赚钱的车企。 |
| **品牌认知** | ★★★ | "智能"标签深入人心，但品牌溢价能力弱于理想和蔚来。 |
| **规模成本优势** | ★★ | 43万辆规模在行业中只能算中等，远不及比亚迪（460万）和特斯拉（~180万）。 |
| **网络效应** | ★★ | 智驾数据积累（里程越多越聪明）有一定网络效应，但需要持续大规模车队在路上跑。 |

### 最大护城河：智能化的"代际领先"

> 小鹏的核心壁垒是 **"全栈自研的AI智驾体系"**——从芯片（图灵）到算法（VLA大模型）到云端算力（万卡集群）到数据飞轮（43万辆车在路上收集数据），这是一条极深的纵向整合链条。同行要么用第三方方案（如华为、Momenta），要么自研能力不及。**但这个护城河能否持续转化为销量和利润，还有待验证。**

### 最大风险

1. **智能化领先优势可能被侵蚀**：华为ADS、比亚迪天神之眼、特斯拉FSD入华，竞争对手都在猛追智驾。小鹏的"技术溢价"窗口期可能只有2-3年。
2. **单车均价持续下探**：主力车型MONA M03售价12万左右，单车均价从2024年的18.8万降至15.9万，品牌向上受阻。
3. **2026年Q1需求骤冷**：Q1指引交付6.1-6.6万辆，同比降30-35%。高增速是否可持续存疑。
4. **竞争白热化**：小米SU7、比亚迪全系降价、华为系车型密集上市，10-25万价位段的厮杀是中国汽车史上最惨烈的。

## 【第五步】生意类型判断

### 主类型：**成长型（即将转为印钞机型？）**

| 维度 | 判断 |
|------|------|
| 核心特征 | 高增速（营收+88%）、亏损快速收窄、经营现金流转正 |
| 所处阶段 | 从"烧钱换规模"到"规模换利润"的拐点期 |
| 最优类比 | **2019年的特斯拉**——Model 3放量前夕，毛利率转正，即将进入盈利轨道 |
| 2025年巨变 | MONA M03像极了当年的Model 3：一款"平民价位+智能化领先"的爆款改变了公司命运 |

### 附加判定：周期型属性

新能源汽车行业有明显的产品周期（新车上市→爬坡→成熟→换代），竞争格局极度动态（今年爆款明年可能滞销）。小鹏要想穿越周期，需要**持续打造爆款的能力**，而非依赖一款车型。

## 【第七步】天花板在哪？

### 三个关键问题

**1. 行业还有多大空间？**

- 中国新能源车渗透率2025年已超50%，增速放缓至20-30%
- 全球新能源渗透率约20%，海外市场空间更大
- 但小鹏的差异化在"智能化"，而不仅是"电动化"，智驾渗透率目前仅5-10%

**2. 市占率还能提升吗？**

| 市场 | 当前表现 | 提升空间 |
|------|---------|:-------:|
| 中国新能源整体 | ~2-3% | 大 |
| 中国新势力阵营 | ~15-16%（第3） | 有空间 |
| 10-15万智驾车型 | 领先 | 先发优势 |
| 海外 | 4.5万辆/年 | **巨大** |

**3. 单客户价值能提升吗？**

- 软件订阅（XNGP智驾包）是潜在增长点，但目前选装率/付费率不高
- 技术输出给大众和其他车企，服务收入增速+66%
- 超快充网络也能贡献长期服务收入

### 结论：中高成长潜力

```
中国市场：空间中等（渗透率已高），市占率提升中
海外市场：空间大，但品牌建设和渠道铺设需时间
软件/服务：空间大，但变现模式尚未跑通
人形机器人：远期期权，短期无贡献
```

> 小鹏的天花板取决于两个"如果"：**如果智能化真的成为消费者选车的首要因素 → 天花板很高；如果智驾变成标配/无差异 → 天花板有限。**

## 【第九步】这家店抗不抗揍？

### 资产负债健康度

```
资产负债率：70.6%（2024年：62.2%，上升）⚠️
有息负债：152.72亿（占总资产14.8%）✅
有息负债率：较低
现金储备：476.6亿
现金/有息负债：3.12倍 ✅ 充裕
经营现金流：+82.6亿（首次年度转正）✅
```

### 负债结构判断

| 负债类型 | 金额 | 占比 | 评价 |
|----------|:---:|:---:|------|
| 贸易及其他应付款 | ~350亿 | 48% | ✅ 好负债（占用供应商资金，证明议价力） |
| 有息借款 | 152.7亿 | 21% | ✅ 可控（现金476亿完全覆盖） |
| 应付票据 | ~80亿 | 11% | ⚠️ 需关注 |
| 其他（租赁、递延收入等） | ~145亿 | 20% | — |

> 表面上资产负债率70.6%看起来很高，但其中有息负债仅152.7亿，大头是占用供应商的应付账款（这是强势车企的典型特征）。**现金储备476亿是有息负债的3倍+，流动性非常充裕。**

### 压力测试

| 情景 | 影响 | 能否承受 |
|------|------|:------:|
| 年交付量腰斩（至20万辆） | 营收降至~400亿，亏损扩大至60-80亿 | ✅ 现金能撑2-3年 |
| 价格战再降2万/辆 | 营收减少85亿，毛利率下降10pct | ⚠️ 亏损会大幅扩大 |
| 融资环境冻结 | 无法发债/增发 | ✅ 自有现金流已转正 |

### 结论：**稳健（表面高杠杆，实质无风险）**

> 小鹏的资产负债表是"看起来吓人、实际上不危险"的典型：70.6%负债率中有近一半是应付账款（好负债），真正需要还本付息的只有152.7亿，而现金有476.6亿。**最大的风险不是财务风险，而是产品力/竞争力风险。**

## 投资亮点与风险总结

### 五大亮点 💡

1. **业绩反转力度惊人**：毛利率1.5%→18.9%，亏损104亿→11亿，经营现金流-20亿→+83亿，三项指标全部在两年内逆转。

2. **中国唯一"靠技术赚钱"的车企**：向大众的技术输出服务收入83.4亿，毛利率约66%，这是传统车企不具备的"软件型收入"。

3. **MONA M03证明小鹏能造爆款**：17.5万辆年销量，在一众竞品中杀出重围，说明公司开始掌握"打造爆款"的方法论。

4. **2025年Q4首次单季盈利**：在研发投入最高（28.7亿）的季度实现盈利，说明盈利不是靠削减研发"省"出来的。

5. **管理层底部增持+与股东利益一致**：AB股架构让何小鹏与股价深度绑定，两次低位增持是真金白银的诚意。

### 五大风险 ⚠️

1. **2026年Q1需求骤降**：Q1指引交付6.1-6.6万辆，同比降30-35%，增速"断崖式"下滑。如果这是趋势性的，估值逻辑要重写。

2. **单车均价持续下探**：从18.8万降至15.9万，品牌"越卖越便宜"，长期可能被定型为"中低端品牌"，与高端化愿景背离。

3. **竞争白热化程度史无前例**：小米SU7、华为智选车、比亚迪全系智驾化，10-25万价位段是中国汽车史上最残酷的战场。

4. **智驾领先优势可能收窄**：华为ADS在快速迭代，比亚迪"天神之眼"在快速追赶，特斯拉FSD随时可能入华。小鹏的技术代差正在缩小。

5. **人形机器人投入是"无底洞"**：2026年AI专项投入计划70亿，其中相当部分投向机器人。这是20年的远期赌注，短期只有成本没有收入。

---

### 综合判断

> **小鹏汽车是2025年中国新能源汽车行业"反转最成功"的故事，没有之一。** 它正处于从"烧钱型成长股"到"盈利型成长股"的关键拐点，有点像2019年的特斯拉——Model 3正在放量、毛利率刚刚转正、单季开始盈利。
>
> 但有几个关键区别：①中国市场竞争远比2019年的美国激烈；②小鹏的技术溢价窗口期可能比特斯拉短得多；③2026年Q1需求降温是一个不可忽视的"减速信号"。
>
> **适合什么样的投资者？** 适合相信"智能化是汽车行业下半场的胜负手"、能承受高波动、愿意用中长期视角（2-3年）来验证逻辑的投资者。不适合追求稳定、低风险或短期收益的投资者。
